Expressions

Expressions are made of the following 5 characters: ψ, (, ), +, and 0. A valid expression is defined as the following:
  • 0 is a valid expression.
  • ψX(Y) is a valid expression, if X and Y are also valid expressions.
  • X0+X1+...+Xn is a valid expression, if all X are also valid expressions.
The 0 in ψ0 can be omitted.

Terms

  • A "plus expression" is an expression of the form described in the third valid expression rule.
  • A "singular expression" is an expression of the form described in the second expression rule.
  • The "psi subscript" of a singular expression ψX(Y) is X.
  • A singular expression expression A is "inside" an expression ψC(B) if A = B, or there exists an n such that the Xn of B = A.

Comparison

For valid expressions A, B, A = B if they are written the same.

A and B are valid expressions.

A = B implies A > B is false.
A > B implies B > A is false.
If B = 0 is true and A = 0 is false, A > B.
If at least one of A or B are not standard, A > B is false.

To compare with a plus expression A with a singular expression B, B > A if B > X0. The same holds for a singular expression A and plus expression B.
To compare A and B both being plus expressions: For all natural numbers n < m such that Xn of A = Xn of B, A > B if Xm of A > Xm of B.

A and B being singular expressions comparison:
  1. If the psi subscript of A > the psi subscript of B, A > B.
  2. For any valid expressions C, D, ψC(A) > ψC(B) ⇔ ψC(ψD(A)) > ψC(ψD(B)).
  3. If there exists a standard substring of A C, the psi subscript of B = the psi subscript of C, and C > B or C = B, A > B.
  4. If checking if ψC(A) > ψC(B), it is true if A > B.
Note that rules can only be used to determine if A > B if you cannot determine whether A > B is true of false with the preceeding rules.

Standardness

0 is standard.
A valid expression E is standard if there are no singular expressions A inside B inside C, such that W > V, X > Y, and Y > Z, with W defined as the least ordinal such that X + W = Y, V being defined as the least ordinal such that Y + V = Z, and X, Y, Z being defined as the values of the psi subscripts of A, B, C. All valid substrings of E must be standard for E to be standard. Additionally, for E being a plus expression, there musn't exist an n such that Xn of E = 0 for E to be standard, and for all Xn and Xn+1 of E, Xn > Xn+1 must be true.

Value

The value of 0 is 0.
E is a standard expression. Let S be the set of the values of all X such that E > X. The value of E is the least ordinal n such that n is not in S.
